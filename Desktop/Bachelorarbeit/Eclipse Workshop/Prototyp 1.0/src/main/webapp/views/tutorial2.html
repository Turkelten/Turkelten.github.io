Tutorial - Funktionen

So funktionierts:

  • Sie können jederzeit vorwärts und rückwärts durch die Anwendung navigieren
  • Einen Startpunkt gibt es nicht
  • Beim Navigieren werden alle Daten die zuvor eingegeben wurden automatisch im Feld gespeichert
  • Das "persmanente" Speichern
    • ist nur auf der Kontrollseite (letzte Seite) möglich
    • schließt ein Feld nach der Bearbeitung ab
    • ermöglicht die Druckausgabe eines Feldes
  • Nach dem "persmanente" Speichern können die Daten immernoch geändert werden
  • Möchten Sie zu einem Späteren Zeitpunkt Weiterarbeiten, gibt es auf der Hauptseite einen Datei Export
  • Laden Sie die Exportierte Datei einfach wieder hoch und Arbeiten Sie weiter
  • Ihre Ausgefüllte Canvas können Sie über die Druckansicht anschauen und sich herunterladen!
